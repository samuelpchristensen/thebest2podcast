# The Best 2 Podcast Website Plan

## Website Structure

### Pages
1. **Home** - Hero, latest episodes, about snippet, subscribe CTAs
2. **Episodes** - Archive with search/filter, transcript links
3. **About** - Sam's story, mission, why this podcast exists
4. **Transcripts** - Individual pages for each episode (SEO gold)
5. **Contact/Subscribe** - Email signup, social links

### Tech Stack
- **Platform:** Static site (GitHub Pages/Netlify) - free, fast, SEO-friendly
- **Generator:** Hugo or Jekyll (simple, fast builds)
- **Transcription:** Whisper API or YouTube auto-captions
- **Domain:** best2podcast.com (or similar)

### SEO Strategy
- Transcript pages = long-tail keyword targeting
- Episode titles as H1s
- Schema.org PodcastEpisode markup
- Sitemap for search engines
- RSS feed integration
